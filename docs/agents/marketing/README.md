# Робоча папка маркетолога Calendary

## Про роль

Маркетолог відповідає за просування календарів Calendary через цифрові канали та аналіз ефективності рекламних кампаній.

---

## Основні обов'язки

### 1. Управління рекламними кампаніями
- Створення та запуск кампаній на Instagram, Facebook, Threads
- Налаштування Google AdSense (опціонально)
- A/B тестування креативів
- Оптимізація бюджету

### 2. Контент-маркетинг
- Планування контент-календаря
- Створення креативів (зображення, відео, тексти)
- Робота з дизайнерами та копірайтерами
- UGC (User Generated Content) кампанії

### 3. Аналітика
- Моніторинг KPI
- Аналіз ефективності каналів
- CAC (Customer Acquisition Cost)
- ROI рекламних кампаній
- Conversion rate optimization

### 4. Промо-акції
- Створення та управління промо-кодами
- Сезонні акції (Новий рік, свята)
- Партнерські програми
- Корпоративні пропозиції

---

## Ключові метрики (KPI)

### Acquisition Metrics
- **Traffic** - відвідувачі сайту
- **Lead Generation** - реєстрації
- **Conversion Rate** - % конверсії в покупці
- **CAC** - вартість залучення клієнта
- **CPC** - вартість кліку
- **CPM** - вартість 1000 показів

### Engagement Metrics
- **CTR** - Click-Through Rate
- **Engagement Rate** - залученість у соціальних мережах
- **Time on Site** - час на сайті
- **Pages per Session** - сторінок за сесію
- **Bounce Rate** - відсоток відмов

### Revenue Metrics
- **Sales** - кількість продажів
- **Revenue** - дохід
- **AOV** - Average Order Value (середній чек)
- **ROAS** - Return on Ad Spend
- **LTV** - Lifetime Value клієнта

### Social Media Metrics
- **Followers Growth** - зростання підписників
- **Reach** - охоплення
- **Impressions** - покази
- **Saves & Shares** - збереження та репости

## Маркетингова стратегія

### Цільова аудиторія

#### B2C (Індивідуальні покупці)
**Сегмент 1: Молоді професіонали (25-35 років)**
- Шукають персоналізовані подарунки
- Активні у соціальних мережах
- Готові платити за якість та унікальність

**Сегмент 2: Батьки (30-45 років)**
- Календарі з сімейними фото
- Подарунки родичам
- Цінують зручність та швидкість

**Сегмент 3: Творчі люди (20-40 років)**
- Власні дизайни та проєкти
- DIY культура
- Високі вимоги до інструментів редагування

#### B2B (Корпоративні клієнти)
- Компанії (календарі з брендингом)
- Маркетингові агенції
- Фотостудії
- Event агенції

### Канали залучення

#### Instagram
**Стратегія:**
- Візуальний контент (приклади календарів)
- Stories з прикладами використання
- Reels з процесом створення
- Колабораціїї з блогерами

**Контент:**
- До/після (як створити календар)
- Testimonials (відгуки клієнтів)
- Behind the scenes (друк, пакування)
- User-generated content

#### Facebook
**Стратегія:**
- Ретаргетинг
- Lookalike audiences
- Групи та спільноти
- Довгоформатний контент

**Контент:**
- Статті про персоналізацію
- Кейси корпоративних замовлень
- Промо-акції
- Конкурси

#### Threads
**Стратегія:**
- Коротка форма контенту
- Storytelling
- Швидкий фідбек від аудиторії

#### Google Ads
**Стратегія:**
- Пошукові кампанії (keywords: "календар на замовлення", "друк календарів")
- Display кампанії (ремаркетинг)
- Shopping кампанії (каталог товарів)

### Сезонність

**Високий сезон:**
- **Листопад-Грудень** - новорічні календарі
- **Вересень-Жовтень** - корпоративні замовлення на новий рік
- **Лютий** - валентинки (календарі з парами)
- **Травень** - подарунки на День матері

**Середній сезон:**
- Весна - весільні календарі
- Літо - сімейні календарі

**Стратегія:**
- Збільшення бюджету у високий сезон
- Контент-підготовка за 1-2 місяці

## Промо-стратегія

### Типи промо-кодів
```
NEWYEAR2025     - 15% на новорічні календарі
FIRST10         - 10% для нових клієнтів
BULK20          - 20% на замовлення 10+ штук
INSTAGRAM15     - 15% для підписників Instagram
VALENTINE2025   - спеціальна пропозиція до Дня Валентина
```

### Партнерські програми
- Блогери (афілейт посилання)
- Фотостудії (комісія 10%)
- Event агенції (корпоративні знижки)

## Робочий процес

### Щотижня
1. **Понеділок** - планування тижня, огляд метрик
2. **Вівторок-Четвер** - створення та запуск контенту
3. **П'ятниця** - аналіз результатів, оптимізація

### Щомісяця
1. Місячний звіт (performance, витрати, ROI)
2. Планування на наступний місяць
3. Брейнсторм нових креативів
4. Аудит конкурентів

### Щокварталу
1. Квартальний аналіз
2. Стратегічне планування
3. Бюджетування
4. A/B тестування нових каналів

## Чек-лист запуску кампанії

### Підготовка
- [ ] Визначено мету кампанії (awareness, consideration, conversion)
- [ ] Визначено цільову аудиторію
- [ ] Створено креативи (мінімум 3-5 варіантів)
- [ ] Написано тексти (headlines, descriptions)
- [ ] Налаштовано трекінг (UTM мітки, pixels)

### Налаштування
- [ ] Вибрано платформу (Instagram, Facebook, Google)
- [ ] Встановлено бюджет та терміни
- [ ] Налаштовано таргетинг
- [ ] Створено A/B тести
- [ ] Налаштовано конверсії

### Запуск
- [ ] Перевірено всі посилання
- [ ] Перевірено landing page
- [ ] Запущено кампанію
- [ ] Встановлено алерти (performance drop)

### Моніторинг
- [ ] Щоденний огляд перші 3 дні
- [ ] Оптимізація по результатам
- [ ] Комунікація з командою
- [ ] Документування результатів
